# RTSP Stream Fixer ![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield]

![Logo](logo.png)

[aarch64-shield]: https://img.shields.io/badge/aarch64-yes-green.svg
[amd64-shield]: https://img.shields.io/badge/amd64-yes-green.svg

This app can be used to fix a RTSP stream which might not be compliant with the RFC. This causes FFMpeg to refuse to play it even though more lenient clients might allow it (e.g. VLC).

See [documentation (Home Assistant)](./documentation) or [documentation (GitHub)](./DOCS.md) for details.
